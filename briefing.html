Своё усиление изображения вместо ContextVision

Памятка к ВКС · позиция, роли, инструмент, пайплайн, сроки · на основе ответов Василия

1. Как подаём (нарратив, 5 шагов)

  1. Где мы сейчас. ContextVision — чужое, покупное, дорогое, юридически зависимое. Каждый снимок мы «арендуем».
  2. Ложный путь — закрываем сразу. «Расшифровать / скопировать» нельзя: патенты, лицензия, и это убьёт сертификацию на выходе на рынок. Проговорить и отмести — показывает зрелость, а не хакерство.
  3. Правильный путь — «чистая комната». Метод не секрет: патенты ContextVision публичны, наука открыта, а современные методы часто лучше старого GOP. Это инженерная задача, не научный прорыв.
  4. Доказательство уже на столе. Открываем демо на реальном снимке Василия: CLAHE (что он пробовал) → наше «Усиление». «Сделано за вечер. Вектор работает».
  5. Роли (см. ниже) — Василий ключевой: валидация и домен на нём.
«Мы не вскрываем ContextVision — это незаконно и тупик. Мы строим своё из открытой науки. И вот доказательство, что это реально — уже на вашем снимке.»

3. Почему Claude Code (в этом контексте, не «вообще»)

  1. Замкнутый цикл «написал → запустил → посмотрел результат → поправил». СС сам гоняет Python/OpenCV на снимке, смотрит выходную картинку, видит ошибку, итерирует. Обработку нельзя «прочитать глазами» — её надо гонять на кадрах. Cursor — больше редактор, цикл висит на человеке.
  2. Читает PDF с картинками — прямой ответ Василию: «курсору надо только текст, PDF переделывать — геморрой». СС читает SDK iRay со схемами напрямую.
  3. Большой репо без «запихивания в контекст». >100к строк: СС сам ходит по файлам, подкачивает нужное — не держит всё в окне (то, чего Василий боялся).
  4. Долгие автономные задачи + субагенты (собрать пайплайн, прогнать на 50 снимках, свести сравнение — пока занят другим).
  5. Память между сессиями (CLAUDE.md / Obsidian) — контекст проекта живёт.
  6. Свежие модели (Opus 4.8) по умолчанию без доплаты за фичу.
Честно: Cursor силён в построчных диффах и план-режиме в редакторе (что Василий любит). Позиция не «выкинь Cursor», а «под обработку изображений — прогон кода на данных + большой репо + PDF SDK — СС нативнее».

4. Из чего собирается «свой enhancement-блок» (всё открытое, легальное)

СтупеньЧто делаетОткрытые кирпичи
1. Препроцессинг14–16 бит, коррекция детектора (offset/gain/битые пиксели — есть с iRay), лог-пространствоNumPy / OpenCV

«Секрет» ContextVision — это ступень 2 (адаптивный денойз), а не контраст. В демо сейчас есть 3+5; добавляем 2 — приближаемся вплотную. Патенты GOP не трогаем — guided filter / Local Laplacian / Noise2Void опубликованы независимо.
